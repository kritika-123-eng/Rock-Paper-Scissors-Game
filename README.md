# ✊ Rock Paper Scissors Game

A responsive Rock Paper Scissors game built using **HTML, CSS, and JavaScript**. Play against the computer, keep track of scores, and reset the game anytime.

## 🚀 Live Demo

👉 https://kritika-123-eng.github.io/Rock-Paper-Scissors-Game/

---

## ✨ Features

- 🎮 Play against the computer
- 📊 Live score tracking
- 🎲 Random computer moves
- 🔄 Reset game functionality
- 📱 Fully responsive design
- ⚡ Built using Vanilla JavaScript

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- DOM Manipulation
